1. Pendahuluan

Tutorial ini mencakup langkah-langkah persiapan awal untuk sebuah Virtual Machine (VM) yang akan difungsikan sebagai Honeypot. Persiapan ini meliputi instalasi beberapa utilitas penting dan pendaftaran VM sebagai agen di server Wazuh untuk pemantauan.

2. Instalasi Utilitas Dasar

Kita akan menginstal beberapa paket dasar yang diperlukan untuk mengelola server dan mengunduh perangkat lunak honeypot nantinya.

Instalasi OpenSSH

Instal OpenSSH server agar VM ini dapat diakses dari jarak jauh.

sudo apt install ssh -y

Aktifkan layanan SSH agar otomatis berjalan saat sistem boot.

sudo systemctl enable ssh

Periksa status layanan SSH untuk memastikan sudah berjalan.

sudo systemctl status ssh

Instalasi cURL

Instal cURL, sebuah alat baris perintah untuk mentransfer data menggunakan berbagai protokol.

sudo apt install curl -y

Tampilkan versi cURL untuk verifikasi.

curl -V

Instalasi Git

Instal Git, yang akan kita perlukan nanti untuk mengkloning repositori perangkat lunak honeypot.

sudo apt install git -y

Tampilkan versi Git untuk verifikasi.

git --version

3. Pemasangan Agen Wazuh

Langkah selanjutnya adalah mendaftarkan VM Honeypot ini ke Wazuh Manager Anda agar aktivitasnya dapat dipantau.

  1. Buka dashboard Wazuh Anda di browser.
  2. Navigasi ke menu "Agents".
  3. Klik pada tombol "Deploy new agent".
  4. Pilih sistem operasi VM Honeypot Anda (misalnya: Ubuntu).
  5. Masukkan alamat IP atau DNS dari Wazuh Manager Anda.
  6. Ikuti petunjuk di layar, terutama langkah 4 dan 5 yang berisi perintah instalasi.

Wazuh akan menyediakan satu baris perintah kustom. Salin dan jalankan perintah tersebut di terminal VM Honeypot Anda. Perintah itu akan mengunduh, menginstal, dan mendaftarkan agen secara otomatis.

4. Verifikasi Koneksi Agen

Setelah skrip instalasi selesai, kita perlu memverifikasi koneksi dari kedua sisi (Agen dan Manajer).

Di VM Honeypot (Agen):

Periksa status layanan wazuh-agent untuk memastikan layanan berjalan tanpa error.

sudo systemctl status wazuh-agent

Di Server VM (Wazuh Manager):

Untuk mengkonfirmasi bahwa agen telah berhasil terhubung dan terdaftar, jalankan perintah berikut di server Wazuh Manager Anda.

sudo /var/ossec/bin/agent_control -l

Jika berhasil, Anda akan melihat VM Honeypot Anda muncul di daftar agen, biasanya dengan status "Active".
