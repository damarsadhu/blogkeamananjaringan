1. Pendahuluan

Docker adalah platform kontainerisasi yang memungkinkan Anda untuk membangun, menguji, dan men-deploy aplikasi dengan cepat. Tutorial ini akan memandu Anda cara menginstal Docker Engine di Kali Linux menggunakan repositori resmi Docker.

Menggunakan repositori resmi memastikan Anda mendapatkan versi terbaru dan pembaruan keamanan tepat waktu.

2. Langkah 1: Update Paket Sistem

Selalu mulai dengan memperbarui daftar paket dan meng-upgrade sistem Anda untuk memastikan semua dependensi terbaru.

sudo apt update sudo apt upgrade -y

3. Langkah 2: Install Paket Pendukung

Instal paket-paket ini agar apt dapat menggunakan repositori melalui HTTPS. Langkah ini opsional namun sangat disarankan.

sudo apt install -y apt-transport-https ca-certificates curl software-properties-common

4. Langkah 3: Tambah Repositori Docker

Selanjutnya, tambahkan kunci GPG resmi Docker dan siapkan repositori Docker. Karena Kali Linux berbasis Debian (saat ini 'bookworm'), kita akan menggunakan repositori Debian.

Tambahkan kunci GPG Docker:

curl -fsSL https://download.docker.com/linux/debian/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg

Tambahkan repositori ke apt:

echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/debian bookworm stable" | \ sudo tee /etc/apt/sources.list.d/docker.list

5. Langkah 4: Install Docker Engine

Update kembali daftar paket Anda untuk menyertakan repositori Docker yang baru, lalu instal Docker Engine.

sudo apt update sudo apt install -y docker-ce docker-ce-cli containerd.io

6. Langkah 5: Jalankan Layanan Docker

Mulai layanan Docker dan aktifkan agar otomatis berjalan setiap kali sistem boot.

sudo systemctl enable docker --now

7. Langkah 6: Verifikasi Instalasi

Periksa versi Docker untuk memastikan instalasi berhasil. Kemudian, jalankan kontainer 'hello-world' untuk mengkonfirmasi bahwa Docker berfungsi dengan baik.

docker --version

Jalankan tes 'hello-world' (perlu sudo saat ini):

sudo docker run hello-world

Anda akan melihat pesan bahwa Docker Anda bekerja dengan benar.

8. Langkah 7: (Opsional) Izinkan Pengguna Non-root

Secara default, hanya root yang dapat menjalankan perintah Docker. Untuk mengizinkan pengguna Anda saat ini menjalankan Docker tanpa sudo, tambahkan pengguna tersebut ke grup docker.

sudo usermod -aG docker $USER

Anda harus Logout dan Login kembali (atau me-reboot) agar perubahan grup ini diterapkan.
